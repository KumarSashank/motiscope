# Target: GSAP (JavaScript)

**Defer to the official GSAP skills** for idiomatic output — do not hand-roll GSAP
guidance here. This file only describes how to feed the spec into them.

## Which GSAP skill to invoke
- **gsap-timeline** — build the sequence. One `tl.to()/from()/fromTo()` per timeline
  entry; use the position parameter for `start_ms` (e.g. absolute `<start_s>` or
  relative `"<+=0.2"`).
- **gsap-core** — individual tweens, `ease` tokens, `duration`, `delay`, `defaults`.
- **gsap-scrolltrigger** — ONLY if the spec has `"scroll_driven": true`.
- **gsap-react** — if the project is React/Next (`useGSAP`, refs, cleanup).
- **gsap-utils** — `gsap.utils` for stagger/interpolate helpers.

## Spec → GSAP inputs
For each `timeline[]` entry:
- `dur_ms / 1000` → `duration`
- `start_ms / 1000` → timeline position
- `ease` → GSAP token via `easing-map.md` (`ease-out`→`power2.out`, `spring`→`back.out(1.7)`, …)
- `props` `{a:[from,to]}` → `fromTo(target, {a:from}, {a:to, ...})`; `y`/`x` map to
  transforms, `opacity`/`scale`/`rotation` directly.
- `stagger.each_ms / 1000` + `from` → `stagger: { each, from }`.
- `hold` segments → a gap in the timeline (`"+=<seconds>"`) or `delay`.

## Minimal runnable harness (`index.html`)
Emit a self-contained page that plays on load so the user can preview immediately:

```html
<!doctype html>
<html>
<head><meta charset="utf-8"><style>
  body { margin:0; display:grid; place-items:center; height:100vh; background:#fff; }
  .card { width:120px; height:120px; background:#3366ff; border-radius:12px; }
  @media (prefers-reduced-motion: reduce){ .card{ opacity:1 !important; transform:none !important; } }
</style></head>
<body>
  <div class="card"></div>
  <script src="https://cdnjs.cloudflare.com/ajax/libs/gsap/3.12.5/gsap.min.js"></script>
  <script>
    // GENERATED BY motiscope — timing/easing measured; sizes/colors estimated.
    const tl = gsap.timeline();
    tl.fromTo(".card",
      { opacity: 0, y: 24, scale: 0.9 },
      { opacity: 1, y: 0, scale: 1, duration: 0.4, ease: "power2.out" }, 0);
  </script>
</body>
</html>
```

Swap the CDN tag for the project's GSAP import when writing into a real app.
